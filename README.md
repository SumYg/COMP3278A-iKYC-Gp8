# WebGUI

## Build
### Windows

Inside Anaconda or just the system environment with Visual Studio. It should be eaiser to install in WSL.
```
python -m pip install -r requirements.txt
```
## Run
``` 
python server.py
```
Then open http://localhost:8080 in browser
